# EASTL-rs
An binary-compatible EASTL implementation written in Rust

[Crate](https://crates.io/crates/eastl-rs) | [Documentation](https://docs.rs/crate/eastl-rs)

## Implemented containers
- `HashMap`
- `HashSet`
- `String`
- `Vector`

## Planned containers
- `HashMultimap`
- `HashMultiset`
- `List`
- `Map`
- `Multimap`
- `Set`
- `Multiset`
